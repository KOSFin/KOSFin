## ⭐ Основные проекты
- **Yookassa-auto-sending-payment-receipts** — сервис для самозанятых: принимает webhook YooKassa, формирует/отзывает чеки в «Мой Налог», ретранслирует события и отправляет уведомления в Telegram.  
  → https://github.com/KOSFin/Yookassa-auto-sending-payment-receipts

- **deepseek-api** — приватный API к Deepseek (V3/R1), поиск, полноценные запросы.  
  → https://github.com/KOSFin/deepseek-api

- **openclaw** — личный ИИ‑ассистент, мультиплатформа.  
  → https://github.com/KOSFin/openclaw

- **2wchat** — мониторинг Twitch‑чата для стримеров: фильтры, экспорт, розыгрыши.  
  → https://github.com/KOSFin/2cwhat.syprod.-

- **notes-ai** — заметки/планировщик с ИИ (в разработке).  
  → https://github.com/KOSFin/notes-ai
